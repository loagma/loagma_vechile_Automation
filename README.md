# First API Project

## Setup

1. Activate virtual environment:
   ```
   venv\Scripts\activate
   ```

2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

3. Run the API:
   ```
   uvicorn main:app --reload
   ```

4. Visit: http://127.0.0.1:8000
   - API docs: http://127.0.0.1:8000/docs
